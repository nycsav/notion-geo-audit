faq
Quick answers.
What is geo-audit?
geo-audit is a free, open-source tool that checks how visible a website is to AI search engines — ChatGPT, Claude, Perplexity, and Gemini. It runs 15 checks on a site's public files and returns a score out of 100 with copy-paste fixes.
Can I run it without installing anything?
Yes — type any URL into the audit box above and press Audit. The 15 checks run in the cloud and your scored report appears in seconds. The CLI, Claude connector, and Notion agent are there when you want more.
Does it work for Notion-hosted sites?
Yes. Audit any *.notion.site address and geo-audit switches to Notion mode: it tells you which fixes you control inside Notion (title, description, indexing, content shape) and which require connecting a custom domain.
What does it cost?
Nothing. MIT licensed and open source — communities can fork it, rebrand it, and run it for their members.
Is my data safe?
Yes. geo-audit reads only files a website already shows the public internet — no login, no API keys for the core audit, no tracking, and nothing stored outside your control.
